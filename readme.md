# Todo App

</div>

## Features

> Add a task to to-do list

>Update a task

>Delete a task

>Show the list of all tasks

## Run Locally

### 1. Clone Repo

```
git clone 
cd TodoApp
```

### 2. activate virtual Env

```
$ env\Scripts\activate
```
### 3. Run file

```
$ py app.py
```
